# YouTube to mp3 downloader and converter

A simple script to download audio stream from YT videos and convert them to mp3


## How to use

Run the main.py, you will be asked for the YT video URL.
The resulting mp3 will be saved in the same directory


## Requirements

Tu run the script, must have installed
- ffmpeg
- ffprobe
from https://www.ffmpeg.org


Python modules (install via pip)
yt_dlp
