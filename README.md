# YouTube to mp3 downloader and converter

A simple script to download audio stream from YT videos and convert them to mp3


## How to use

Run the yt_to_mp3.py, you will be asked for the YT video URL.
The resulting mp3 will be saved in the same directory


## How to use m4a to mp3 script

Copy m4a file to be cnverted into this directory and Run following comand
`python3 m4a_to_mp3.py [filename.mp3]`
The converted mp3 file will be saved in this directory, using original name

Tu run the script, must have installed
- ffmpeg
- ffprobe
